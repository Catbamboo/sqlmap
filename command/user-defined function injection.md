# 10. 用户自定义函数注入

User-defined function injection

    These options can be used to create custom user-defined functions
    这些选项可用于创建自定义用户定义的函数
    
    --udf-inject        Inject custom user-defined functions
    					注入自定义用户定义的函数
    --shared-lib=SHLIB  Local path of the shared library
    					共享库的本地路径
